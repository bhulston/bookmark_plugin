Obsidian Bookmarks

Setup Your Bookmark Settings

If you have been looking for a better alternative to Chrome bookmarks to keep track of your articles and videos, you've come to the right place! This requires Obsidian to be used (maybe I will add more integrations in the future!). \n I've tried to make this extension functional and customizable so that you can organize your notes as you like. There is a default included though. \n Note: this requires the Obsidian Local REST API to work (as that is how we create new notes in your vault!). So make sure that is working as well.

Features

  • Obsidian File Search: Choose bookmark locations easily using search in extension
  • Automatically Collect DOM data from webpage (Optimized for Medium and Youtube!)
  • Exclude files and folders you don't want to show show up in the sidebar search
  • Add additional context and create new notes in vault corresponding with bookmark

Privacy

This extension communicates with your Obsidian vault via an API and stores no data other than the settings below...
