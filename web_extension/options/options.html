Obsidian Bookmarks

If you have been looking for a better alternative to Chrome bookmarks to keep track of your articles and videos, you've come to the right place! This requires Obsidian to be used (maybe I will add more integrations in the future!).


Usage

I've tried to make this extension functional and customizable so that you can organize your notes as you like. There are default MD and YAML templates in the GitHub Repo.


Note: this requires the Obsidian Local REST API to work (as that is how we create new notes in your vault!). You can find more information on that here


What Can It Do?

  • Obsidian File Search: Choose bookmark locations easily using search in extension
  • Automatically Collect DOM data from webpage (Optimized for Medium and Youtube!)
  • Exclude files and folders you don't want to show show up in the sidebar search
  • Add additional context and create new notes in vault corresponding with bookmark

Privacy

This extension communicates with your Obsidian vault via an API and stores no data other than the settings below...
